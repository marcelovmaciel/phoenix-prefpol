# Preferences.jl

`Preferences.jl` provides formal tools for working with finite preference data:
candidate pools, strict and weak rankings, profiles and weighted profiles,
pairwise comparison, consensus rankings, majority support, single-peakedness,
and preference-space polarization.

It is the reusable formal API in this monorepo. The applied `PrefPol` pipeline
should be documented separately.
